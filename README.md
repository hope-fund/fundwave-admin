# FundWave Admin Dashboard

The secure administrative control portal and financial operations engine for the FundWave Crowdfunding Platform.

## Core Features
- **Admin Auth & Security:** Role-based access control (RBAC) via secure JWT handling.
- **Donation Verification:** Manual bank transfer reconciliation and confirmation flow.
- **Ledger Audit:** Immutable system of record for all platform financial events.
- **Campaign & Payout Management:** Structural approvals for fundraising lifecycles and beneficiary withdrawals.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Axios
